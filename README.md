# unicode-emoji

A simple Go package that allows you to list all available emojis in the unicode space.

## Dependencies

* Go 1.16

## Building & Running

A simple `make build` will do. Note that this requires `make`, `curl`, and `sed` to work properly.
Running this will also download the required unicode code points from unicode.org (requires an internet connection)

As for running you can run an example in the `examples` directory:

```
go run examples/printer.go
```

## Example usage

Below you can find simple usages of the emoji package.


### Print all emojis
This is also how all the emojis below were added to the README

```go
package main

import (
	emoji "edholm.dev/unicode-emoji"
	"fmt"
	"log"
)

func main() {
	emojis := emoji.NewEmojis()
	all, err := emojis.All()
	if err != nil {
		log.Fatalln(err)
	}

	for i, e := range all {
		if i%50 == 0 {
			fmt.Printf("\n")
		}
		fmt.Printf("%s", e.String())
	}
}
```
